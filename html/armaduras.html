Armaduras

¿Cómo funcionan las armaduras?

En Monster Hunter, las armaduras no solo sirven para aumentar la defensa del cazador. Cada pieza puede aportar resistencias elementales, habilidades especiales y ranuras para decoraciones.

Un set completo normalmente está formado por cinco piezas: casco, pechera, guantes, cintura y botas. Combinar correctamente estas piezas permite crear armaduras enfocadas en ataque, defensa, recolección, evasión o resistencia elemental.

Partes de una armadura

Casco

Protege la cabeza y suele aportar habilidades ofensivas o de utilidad.

Pechera

Es una de las piezas principales y normalmente aporta buena defensa base.

Guantes

Pueden mejorar habilidades relacionadas con ataque, afinidad o manejo de armas.

Cintura

Complementa el set y ayuda a equilibrar habilidades y resistencias.

Botas

Suelen aportar habilidades de movilidad, evasión o resistencia.

Simulador de set

Selecciona una pieza de cada tipo para calcular la defensa total, resistencias elementales y habilidades del conjunto.
